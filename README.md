## Description

[Nest] **NestJS Task Management Application**

## Installation

```bash
$ npm install
```
$ npm install -g @nestjs/cli
$ nest --version


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Create new module

$ nest g module name_of_the_module

## Create new controller

$ nest g controller name_of_the_module --no-spec
